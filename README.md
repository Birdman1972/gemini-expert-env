# Gemini CLI Expert Environment Setup

This repository contains the configuration needed to instantly upgrade Gemini CLI into the **"Full Form" Privileged Expert Mode** (墨鏡模式 |⌐■_■|).

## Usage

To automatically configure a new environment, simply ask Gemini CLI:

> "Please read the `birdman1972/gemini-expert-env` repository and follow its instructions to upgrade my environment."

Alternatively, run the installation script directly:
```bash
bash install.sh
source ~/.bashrc
```
